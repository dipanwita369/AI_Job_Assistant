An automated Python-based job tracking system that fetches the latest internship opportunities in India using Google Jobs API, prioritizes top companies, filters recent postings, and notifies users in real time.

## Features
- Fetches Data, Business, Finance & Analytics internships
- Filters jobs posted within last 30 days
- Prioritizes top companies
- Avoids duplicate job listings
- Stores data in Excel
- Desktop notifications for new jobs

## Tech Stack
- Python
- SerpAPI (Google Jobs API)
- Pandas
- Requests
- Excel Automation
- Windows Notifications

## How It Works
1. Fetches job listings using Google Jobs API
2. Cleans and filters recent job data
3. Assigns priority scores to companies
4. Stores results in Excel
5. Sends notification when new jobs are added
